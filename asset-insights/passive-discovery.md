    # Passive discovery

    Passive discovery pages cover creating passive discovery, editing passive discovery, and using it as a source when active scans or API integrations are not the best fit.

## When to use passive discovery

* You need observational asset visibility without active scanning.
* Asset signals already flow through network telemetry.
* You want to compare passive findings with managed IPAM or cloud inventory.
