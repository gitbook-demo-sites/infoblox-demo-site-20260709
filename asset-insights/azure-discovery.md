    # Azure discovery

    Azure discovery content covers general setup, destination configuration, excluded asset types, review, credential creation, permissions, and limitations.

## Azure setup checklist

* Create Microsoft Azure credentials.
* Define destination and discovery scope.
* Exclude irrelevant asset types.
* Review limitations and expected asset coverage.
* Validate discovered assets in the monitoring and data-table views.
