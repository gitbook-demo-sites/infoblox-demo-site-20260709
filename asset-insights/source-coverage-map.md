    # Source coverage map

    This page maps the original Universal Asset Insights source space into the curated demo navigation. The source sitemap lists 155 pages for this area, so the demo consolidates repetitive leaf tasks into grouped GitBook pages rather than creating hundreds of shallow pages.

## Coverage families

| Source family | Where it lands in this demo |
| --- | --- |
| Architecture and concepts | About, architecture, asset profiles, discovery sources, asset definitions, and insight classifications. |
| Cloud discovery | AWS, Azure, GCP setup, destinations, exclusions, credentials, permissions, summaries, reviews, and limitations. |
| On-prem discovery | NIOS-X discovery, Network Insight, network discovery services, passive discovery, IPAM resources, and diagnostics. |
| Microsoft discovery | Microsoft servers, Active Directory, DNS, DHCP, Event Log, WEC, Universal Agent, prerequisites, and service locations. |
| Third-party discovery | CrowdStrike, Akamai, Cloudflare, ServiceNow, Juniper Mist, Cisco Catalyst SD-WAN, and provider credentials. |
| Operations and reports | Asset Explorer, data tables, monitoring workspaces, security workspaces, tags, export, managed/unmanaged assets, and detailed reports. |
