    # Network Insight

    Network Insight content includes general settings, credentials, service ports, discovery services, polling configuration, object scope, summary, advanced settings, and network discovery edit/delete flows.

## Configuration areas

| Area | Purpose |
| --- | --- |
| Credentials | Authenticate to network infrastructure and discovery targets. |
| Service ports | Define reachable services and scan boundaries. |
| Polling | Control cadence and freshness. |
| Object scope | Limit discovery to relevant infrastructure. |
| Advanced settings | Tune behavior for complex networks. |
