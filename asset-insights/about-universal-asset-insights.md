    # About Universal Asset Insights

    Universal Asset Insights brings discovered infrastructure into a shared asset model. It helps teams identify assets, reconcile unmanaged resources, understand risk context, and connect discovery findings back to network and security operations.

## What it unifies

| Domain | Examples |
| --- | --- |
| Cloud | AWS, Azure, Google Cloud resources and metadata |
| Network | NIOS-X discovery, Network Insight, passive discovery, DDI relationships |
| Microsoft | Active Directory, DNS, DHCP, Event Log, Universal Agent inputs |
| Security and edge | CrowdStrike, Akamai, Cloudflare, ServiceNow, network integrations |
