    # Source coverage map

    This page maps the original Universal DDI source space into the curated demo navigation. The source sitemap lists 200 pages for this area, so the demo consolidates repetitive leaf tasks into grouped GitBook pages rather than creating hundreds of shallow pages.

## Coverage families

| Source family | Where it lands in this demo |
| --- | --- |
| What's new and licensing | What's new, licensing, entitlements, supported versions, and release-note-style updates. |
| DNS | DNS views, zones, records, ACLs, cache, transfers, DNS profiles, and forwarding. |
| DHCP | Global DHCP properties, config profiles, ranges, reservations, options, fingerprints, and thresholds. |
| IPAM | IP spaces, subnets, address blocks, discovered resources, utilization, and metadata. |
| Hybrid services | NIOS-X, NIOS, HA groups, Anycast, appliances, connectivity, and reporting server setup. |
| Cloud and Microsoft | AWS Route 53, Microsoft Active Directory, Microsoft DNS/DHCP, and credential workflows. |
| Operations | Reports, imports, exports, data connector, log export, service integrations, and troubleshooting. |
