    # AWS Route 53

    Universal DDI includes Amazon Web Services integration and Route 53-specific workflows such as credentials, permissions, limitations, creating credentials, modifying credentials, and deleting credentials.

## Route 53 operating model

* Add AWS credentials with the required Route 53 permissions.
* Discover or manage DNS zones in the context of the wider DDI model.
* Review documented limitations before treating cloud DNS as equivalent to on-prem authoritative DNS.
* Align credential lifecycle with security and cloud platform ownership.

See [Public cloud DNS](cloud-dns.md) for the multi-provider view.
