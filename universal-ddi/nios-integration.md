    # NIOS integration

    Universal DDI documentation includes NIOS integration topics such as configuring NIOS in the Infoblox Portal, NIOS reporting, NIOS DNS query responses, NIOS source configuration, and data connector registration.

## Integration surfaces

* Register NIOS and related source configurations.
* Configure NIOS DNS query and response capture.
* Send DNS, DHCP, RPZ, and IPAM metadata to downstream reporting pipelines.
* Use NIOS reporting as a destination for selected telemetry.

See [Data connector and log export](data-connector-log-export.md) for the export side of this workflow.
