    # IPAM and address management

    IPAM pages organize network space, discovered resources, address utilization, and lifecycle operations for subnets and blocks.

## Main objects

| Object | Purpose |
| --- | --- |
| IP spaces | Separate overlapping or delegated address domains. |
| Address blocks | Reserve larger ranges for planning and allocation. |
| Subnets | Manage routable networks and operational ownership. |
| Ranges and reservations | Control DHCP allocation, fixed addresses, and exceptions. |
| Discovered resources | Reconcile managed IPAM with observed infrastructure. |
