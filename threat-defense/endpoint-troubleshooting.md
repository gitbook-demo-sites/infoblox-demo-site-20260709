    # Endpoint troubleshooting

    Endpoint troubleshooting spans status checks, endpoint health checks, protected bypass mode, compatibility guidelines, service logs, DFP logs, and endpoint group state.

## Triage checklist

* Confirm endpoint version and compatibility.
* Check endpoint status and health.
* Validate endpoint group assignment and policy scope.
* Review DNS forwarding path and internal domain handling.
* Use service logs and security events to trace behavior.
