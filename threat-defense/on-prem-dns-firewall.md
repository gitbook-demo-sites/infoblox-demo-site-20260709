    # On-prem DNS Firewall

    On-prem DNS Firewall content includes DNS Firewall, on-prem DNS Firewall service, RPZ feed access, appliance sizing, Trinzic guidance, and feed availability questions.

## When it matters

* DNS enforcement needs to remain close to on-prem recursive infrastructure.
* RPZ or DNS Firewall feeds are part of the customer's existing security architecture.
* Appliance sizing and service prerequisites determine the rollout path.
* Hybrid deployments need both cloud-delivered and on-prem controls documented side by side.
