    # Source coverage map

    This page maps the original Threat Defense source space into the curated demo navigation. The source sitemap lists 200 pages for this area, so the demo consolidates repetitive leaf tasks into grouped GitBook pages rather than creating hundreds of shallow pages.

## Coverage families

| Source family | Where it lands in this demo |
| --- | --- |
| What's new, licensing, API | Release notes, subscriptions, RPZ access questions, API getting started, and appendices. |
| Security policy | Policies, rules, precedence, filters, custom lists, geolocation, bypass codes, and network scopes. |
| Threat intelligence | TIDE, Dossier/research, active indicators, feeds, data profiles, InfoRanks, and threat classification. |
| Endpoints and DFP | Endpoint lifecycle, endpoint groups, mobile enrollment, DFP, internal domains, recursive queries, and DoH. |
| Reports | Executive, comprehensive, summary, security activity, DNS activity, web content, users, devices, and events. |
| Operations | Data connector, SIEM integrations, RBAC, service logs, notifications, webhooks, audit logs, and troubleshooting. |
