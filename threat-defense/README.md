# Threat Defense

Infoblox Threat Defense is a hybrid cybersecurity solution that uses DNS to detect, block, investigate, and report on threats before they cause impact. It combines DNS Firewall, DNS forwarding, endpoints, threat intelligence, policy, research, reporting, and integrations into one operational workflow.
