    # API getting started

    The source docs include an Infoblox API Getting Started Guide and an appendix for Threat Defense API guidance.

## Automation surfaces

| Area | Example use |
| --- | --- |
| Policy automation | Create, update, and audit security-policy settings. |
| Feeds and lists | Manage custom TIDE feeds, custom lists, and threat intelligence workflows. |
| Reporting | Retrieve events and summarize security posture. |
| Platform operations | Use service API keys and RBAC-controlled access. |

A production migration should use generated API reference pages where an OpenAPI specification is available.
