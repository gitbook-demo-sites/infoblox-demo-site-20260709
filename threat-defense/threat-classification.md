    # Threat classification

    Threat classification docs include the Threat Classification Guide, Infoblox InfoRanks, Threat View, lookalike domain monitoring, excluded bogons, active indicators, and confidence scoring.

## Classification concepts

| Concept | Why it matters |
| --- | --- |
| Threat class | Communicates what type of risk is being observed. |
| Threat level | Helps prioritize response and policy action. |
| Confidence score | Signals how reliable an indicator or classification is. |
| Lookalike domains | Highlights impersonation and brand-abuse risk. |
